## 快速开始

项目运行需要`NodeJS`，请自行安装运行环境。

> 推荐使用pnpm，也可使用其他依赖管理工具，如：npm、yarn、cnpm等

```bash
# 安装依赖
pnpm install
# 运行项目
pnpm dev
```

项目运行后会自动打开本机的Chrome浏览器，并自动加载插件代码。

项目核心框架为[WXT](https://github.com/wxt-dev/wxt)，具体可见：[WXT文档](https://wxt.dev)
